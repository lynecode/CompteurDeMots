# CompteurDeMots

# Objectif
Ce projet est une application Java qui lit un fichier texte (texte.txt)
et affiche le nombre total de mots qu'il contient.

Importez le projet dans IntelliJ IDEA.
Confirmez que le JDK est correctement paramétré en version 21.
Vérifiez la présence du fichier texte.txt à la racine du projet.
Exécutez la classe principale.
